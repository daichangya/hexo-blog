注意事项

  1. API 密钥：所有请求需在 Header 中传递 Authorization: Bearer $OPENAI_API_KEY。
  2. 费用：不同接口的计费方式不同，需参考 OpenAI 定价页。
  3. 模型支持：部分接口仅限特定模型（如 Whisper 仅支持 whisper-1）。
  4. 流式响应：若需流式传输（如逐字输出），可添加 "stream": true 参数。
